# 1A2B 猜數字遊戲

這是一個經典的 1A2B 猜數字遊戲的文字介面版本。

## 遊戲規則

- 電腦會隨機產生一個 4 位數字（數字不重複）
- 玩家需要猜測這個數字
- 每次猜測後，系統會回應：
  - **A**：數字和位置都正確的個數
  - **B**：數字正確但位置錯誤的個數
- 每局遊戲最多可以猜 10 次
- 猜對或用完次數後，可以選擇是否重新開始

## 如何執行

### 方法一：使用 uv（推薦）

本專案使用 uv 管理 Python 環境（Python 3.13）：

```bash
uv run python 1a2b.py
```

首次執行時，uv 會自動下載 Python 3.13 並創建虛擬環境。

### 方法二：直接執行

如果你已安裝 Python 3.6 或以上版本：

```bash
python 1a2b.py
```

## 遊戲範例

```
第 1 次猜測：1234  →  1A2B
第 2 次猜測：1567  →  2A0B
第 3 次猜測：1589  →  4A0B
```

> **注意**：實際遊戲中，A 會以綠色顯示，B 會以黃色顯示，讓結果更清晰易讀！
