# 1A2B 猜數字遊戲

一個經典的文字模式猜數字遊戲（又稱 Bulls and Cows）。

## 遊戲規則

- 電腦會隨機產生 4 個不重複的數字（0-9）
- 玩家需要猜測這 4 個數字
- **A** 代表數字和位置都正確
- **B** 代表數字正確但位置錯誤
- 每局遊戲最多可猜 10 次
- 猜對了顯示成功訊息，超過 10 次顯示失敗訊息

## 遊戲範例

```
第  1 次猜測: 1234  →  1A2B
第  2 次猜測: 1256  →  2A1B
第  3 次猜測: 1278  →  4A0B
```

表示：
- 第 1 次：有 1 個數字位置正確，2 個數字存在但位置錯誤
- 第 2 次：有 2 個數字位置正確，1 個數字存在但位置錯誤
- 第 3 次：全部猜對！(4A0B)

## 如何執行

### 方法 1：使用 uv 執行（推薦）

本專案使用 uv 管理 Python 環境（Python 3.13）。

```bash
# 首次執行需要同步環境
uv sync

# 執行遊戲
uv run python 1a2b.py
```

### 方法 2：使用系統 Python 執行

```bash
python 1a2b.py
```

## 系統需求

- Python 3.13 或以上版本
- uv（推薦使用）

## 特色功能

✅ 輸入驗證（檢查是否為 4 位不重複數字）  
✅ 結果顯示在猜測數字的右側  
✅ 最多 10 次猜測機會  
✅ 遊戲結束後可選擇重玩  
✅ 友善的使用者介面  
✅ 完整的中斷處理（Ctrl+C）
